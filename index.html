Getting started

Don´t be shy, read the tiny text and you will be able to use Asana Time Track without any problems. 

The Tool will only fetch tasks assigned to you. This is because otherwise you will get tons of tasks you do not need.
An important thing additionally is, that you have to archive your task from your "My Tasks" view.
Otherwise Asana will not recognize this tasks as archived and will fetch it instead.
Maybe this changes in the future, but right now there is a difference between archiving a task in the project view
and your personal task view.

Please remember the Asana Api is not as fast as a Porsche, so take your time when the tool fetches
the data from the servers. Also you have to know that this tool is working only with an
server (e.g. apache), because it´s using php.

All you have to know about Asana-Modification!

1. // WHERE ARE YOUR ASSIGNED TASKS

It´s quite easy, all your personal assigned tasks are in the "My Tasks" section.

2. // ASANA TIME TRACK MANIPULATES TASKNAMES

Asana Time Track works fine with already existing Tasks. Every single tracked task gets automatically something like this [ET: 0h 0m] [WT: 0h 0h] attached at the end.

ET = Estimated Time
WT = Worked Time

You can also manipulate all ETs and WTs manually. ATTENTION! The string-format has to be like
[ET: 0h 0m] [WT: 0h 0h] and everything is working fine. At least we hope so ;-)

And now you are ready to use the Asana Time Track tool!

If you need more informations, just follow the next explanation steps...

1. // ASANA-API-KEY

First of all you have to place your Asana-Api-Key into the upcoming modal-box.
Where do i get my api-key? Just click here to read the documenation or go directly to the api-key-tab.
The Asana-Api-Key is only saved on your computer (Cookie).

2. // YOUR WORKSPACES

After that your workspaces are loading automatically. Please choose one workspace.
We excluded the Personal Workspace, because it troubled the tool much, also we think you dont need to track time in your spare time :)


3. // YOUR ASSIGNED TASKS

Your assigned tasks will be shown.
Only assigned tasks!

4. // IT`S TIME TO TRACK AUTOMATICALLY

Press "Start" an the time-tracker starts. Asana Time Track saves the "Worked Time" automatically into the Taskname every minute.

It looks like:
Your taskname [ET: 1h 5m] [WT: 0h 5m]

5. // MANUAL TIME TRACKING IS POSSIBLE TOO

Your are able to change "Estimated Time" and "Worked Time" manually.
All changes are saved automatically.
